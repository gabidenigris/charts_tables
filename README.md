# Charts & Tables (PT/BR)

Repositório de gráficos e tabelas autorais produzidas no contexto de análises econômicas e pareceres técnicos, usando microdados brasileiros. Os scripts são escritos em Python e R, e os outputs exportados como `.png` para visualização.



## Estrutura do repositório

```
charts_tables/
├── README.md
├── graficos/
│   └── projeto_1/
│       ├── script_projeto_1.py
│       └── output/
│           ├── grafico_1.png
│           └── grafico_2.png
└── tabelas/
    └── projeto_2/
        ├── script_projeto_2.py
        └── output/
            └── tabela_1.png
            └── tabela_2.png
```
