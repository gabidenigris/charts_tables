# Charts & Tables (PT/BR)

Repositório de visualizações e tabelas de dados autorais produzidas no contexto de análises econômicas e pareceres técnicos. Os scripts são escritos em Python e R, e os outputs exportados como `.png` para visualização.



## Estrutura do repositório

```
charts_tables/
├── graficos/
│   ├── *.ipynb          # Scripts de visualização
│   └── output/          # PNGs exportados
└── tabelas/
    ├── *.ipynb          # Scripts de tabelas
    └── output/          # PNGs exportados
```
